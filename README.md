#  E-commerce Data Analysis Project (EDA)

## Project Overview
This project provides a comprehensive **Exploratory Data Analysis (EDA)** for an E-commerce platform. It covers everything from data cleaning and referential integrity to customer behavior and sales forecasting.

## Key Features
- **Data Integrity:** Verified 10 Foreign Key relationships.
- **Customer Insights:** Identified VIP customers and demographics (Age/Gender).
- **Sales Analysis:** Analyzed $6.3M in revenue across top US cities.
- **Logistics:** Evaluated shipping performance (FedEx vs UPS).
- **Market Basket Analysis:** Discovered product bundling patterns.

## Tools Used
- **Python** (Pandas, NumPy)
- **Visualization:** Matplotlib, Seaborn
- **Statistics:** Scipy (Skewness & Kurtosis analysis)

## Final Summary
- **Total Revenue:** $6,392,434
- **Top Category:** Electronics (68% of sales)
- **AOV:** $639.24
